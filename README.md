# 小小调度员

一个面向一年级小朋友、并为二年级做准备的铁路主题互动 Web 项目。题库按 60 天生成，每天 10 道不重复题，覆盖语文、数学、英语启蒙和综合应用；火车背景知识包含 CR450 新一代动车组、复兴号 CR400、东风内燃机车和电力机车。

## 运行方式

直接在浏览器打开 `index.html` 即可运行，不需要安装依赖。

## 内容原则

- 火车知识可以偏专业，满足喜欢铁路的小朋友。
- 学科难度以一年级为主，加入少量二年级预备：拼音、部首、量词、通顺句、100 以内加减、比多少、两步应用题。
- 英语按非母语零基础儿童重组：先做字母形状识别、大小写匹配、首字母意识、词图匹配、听读对应和固定短句，不直接要求背单词。
- 题型范围参考国家中小学智慧教育平台、义务教育课程标准、Cambridge English Pre-A1/Starters 和 British Council 少儿英语启蒙建议；题目为原创生成，不复制第三方题库原题。
- 60 天内每天 10 题，题干不重复。
- 每题只有一次作答机会，作答后锁定，不能重做当天题目。
- 错题本保存在本地浏览器，记录日期、科目、薄弱项、车型、题目、孩子答案、正确答案和提示。
- 错题本支持导出 JSON，便于后续分析薄弱项。
